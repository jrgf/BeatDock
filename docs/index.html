Why Choose BeatDock?

Rich Music Playback

Play songs and playlists from YouTube with queue management, shuffle, volume control, and more.

Modern Slash Commands

Utilizes Discord's fast and intuitive slash commands for a seamless user experience.

Multi-Language Support

Comes with English and Spanish translations out of the box, and can be easily extended.

One-Command Deployment

Get up and running in seconds with the power of Docker and docker compose.

Role-Based Permissions

Control who can use the bot with a simple, role-based permission system.

Stateless by Design

No database required. All state is kept in memory, perfect for containerized setups.

Command Reference

A complete list of all available slash commands.

Command Description
/play <query>Play a song or playlist (searches YouTube if not a URL)
/skipSkip the current song
/stopStop playback and clear the queue
/pauseToggle pause/resume
/queueShow the current queue
/nowplayingShow info about the current song
/shuffleShuffle the queue
/volume <0-100>Set playback volume
/clearClear the entire queue
/backPlay the previous track

Quick Start Guide

Deploying BeatDock is incredibly simple with Docker.

1. Clone the Repository

Get the source code on your machine.

git clone https://github.com/lazaroagomez/BeatDock.git
cd BeatDock

2. Configure your Bot

Create a .env file with your bot token and client ID.

 # Required
   TOKEN=your_discord_bot_token
   CLIENT_ID=your_discord_client_id

   # Lavalink connection
   LAVALINK_HOST=lavalink
   LAVALINK_PORT=2333
   LAVALINK_PASSWORD=youshallnotpass

   # Optional: Language (default is "en")
   DEFAULT_LANGUAGE=en

   # Optional: Timeout settings (milliseconds)
   QUEUE_EMPTY_DESTROY_MS=30000  
   EMPTY_CHANNEL_DESTROY_MS=60000 

   # Optional: Permission Settings (comma-separated role IDs)
   ALLOWED_ROLES=

3. Deploy Commands

Run this one-time command to register the slash commands with Discord.

docker compose run --rm bot npm run deploy

4. Start the Bot

That's it! Your bot is now online.

docker compose up -d
